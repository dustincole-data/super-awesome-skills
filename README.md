# Super Awesome Skills

Public-facing skills you can load into your own agent environment and use.

## Skills

- `deep-research/` — external deep research loop with query angles, discovery rounds, gap analysis, source ranking, and cited dossiers.
- `write-like-you/` ... fill-in-once voice guide built from your own real writing, plus a banned-AI-tells self-check, so an agent drafts and edits in your voice. See its README.
- `write-like-you-tune/` ... the paired improvement loop: hand it an AI-draft/your-rewrite pair and it folds the durable rule back into your voice guide, gated on your approval.
